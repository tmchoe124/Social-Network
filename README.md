# Social-Network

## Description

This project is a **Social Network API** built with **Node.js**, **Express.js**, and **MongoDB** (via the **Mongoose** ODM). It allows users to create and share thoughts, react to friendsâ€™ thoughts, and manage a friend list. The application demonstrates RESTful routes for all CRUD operations and includes functionality to add and remove reactions and friends.

## Installation

1. Clone the repository 
    ```bash
    git@github.com:tmchoe124/Social-Network.git
    ```
2. Navigate to the project folder
  ```bash
  cd Social-Network
  ```
3. Install dependencies
    ```bash
    npm install
    ```
4. Start server
    ```bash
    npm start
    ```
